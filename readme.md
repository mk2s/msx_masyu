this is a MSX game written in H-Forth (link_here)  The source generates a .com file which can be packages into a DSK file and run in emulators.  The program should run on MSX1 as well as any later MSX.

The program was written to submit to MSXDEV25

In order to compile the source you'll need to setup a MSX-DOS disk and copy HFORTH.COM from H-FORTH site.  Then HFORTH masyu.4th will compile and produce a masyu.com file.

How to play the game.
move around the board.  Hold shif to connect cells, hold ctl to disconnect.
